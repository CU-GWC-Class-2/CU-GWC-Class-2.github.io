FALL 2017

We started some advanced CSS this week, learning how to make simple shapes (triangles, squares, and circles) out of divs and then how to animate them!! To make squares out of divs, we created a .square css class and defined two properties: height and width. To make circles, we used the border-radius property in a .circle css class, setting it equal to the same value we chose for height and width (what happens when we don't make border-radius the same size as the height and width of the div??). To make triangles, we created a .triangle css class and set the properties of height and width to 0. Then, since the borders of a div are defined by triangles by default, we made all of the borders of the div transparent except for one (which we gave a specific color), using the properties border-right, border-left, border-top, and border-bottom.

When we made the shapes move, we defined animations in our css file with @keyframes. We defined exactly where we wanted our element to be at a given point in the animation (0%, 25%, 66%, etc) by altering multiple properties at each point. To get this animation to work for a specific element, we gave the element an id and defined its animation property with the name of the animation we created with @keyframes and the duration of the animation. We even manipulated the repetition of the animation, specifying whether we wanted it to be infinitely repeating and/or alternating directions!

ANIMATION WITH CSS NOTES

This week in class we learned how to make a navigation bar two different ways! For our first nav bar, we used some of the basic HTML and CSS we learned in the last two weeks to make it simple and elegant. For our second nav bar, we used Boostrap 3, a CSS Framework that allows you to use pre-styled CSS classes for commonly used elements in webpages (such as buttons, links, or nav bars!) to give our navigation some fancy capabilities, such as drop down menus and collapsing into a button when the window size shrinks (like for a phone).
